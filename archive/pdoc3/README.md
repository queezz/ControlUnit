# pdoc3 Archive

This directory contains the legacy API documentation generated by
[pdoc3](https://pdoc3.github.io/pdoc/) for the `controlunit` package.

These files were generated with:

```sh
python docs/make.py
# which ran: pdoc controlunit -o ./docs/ --html --force
```

The HTML was last regenerated on 2023-06-22 and committed as static artifacts
(see git log). They have been moved here to make way for the MkDocs Material
documentation infrastructure introduced in 2026.

The `make.py` build script is preserved alongside the output it produced.

## Contents

| File | Module |
|---|---|
| `index.html` | package index |
| `AIO.html` | AIO ADC board driver |
| `main.html` | application entry point |
| `worker.html` | background worker thread |
| `tc.html` | thermocouple interface |
| `components/` | UI widget submodules |
| ... | other modules |
